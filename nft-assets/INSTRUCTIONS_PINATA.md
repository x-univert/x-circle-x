# Instructions pour uploader sur Pinata IPFS

## Étape 1: Créer un compte Pinata gratuit

1. Allez sur https://app.pinata.cloud/
2. Cliquez "Sign Up"
3. Créez un compte gratuit (1 GB de stockage gratuit)

## Étape 2: Uploader les GIFs

1. Connectez-vous à votre compte Pinata
2. Cliquez sur "+ Add Files" puis "Folder"
3. Naviguez vers le dossier `nft-assets` de ce projet
4. Sélectionnez TOUS les fichiers GIF (0.gif à 12.gif)
5. Nommez le dossier "xcirclex-nfts"
6. Cliquez "Upload"

## Étape 3: Copier le CID

Après l'upload, Pinata vous donnera un CID (Content Identifier).
Il ressemble à: `QmXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX` ou `bafybeXXXXXXXX`

## Étape 4: Tester les URLs

Testez que vos images sont accessibles:
- https://gateway.pinata.cloud/ipfs/VOTRE_CID/0.gif
- https://ipfs.io/ipfs/VOTRE_CID/0.gif

## Étape 5: Me donner le CID

Une fois que vous avez le CID, donnez-le moi et je mettrai à jour le contrat NFT.
